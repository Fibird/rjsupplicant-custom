使用说明：
	解压后，使用终端进入当前根目录，运行rjsupplicant.sh脚本即可启动RG-SU。
	如果rjsupplicant.sh 脚本无法运行（非可执行文件），请运行以下命令: "sudo chmod +x ./rjsupplicant.sh"
	第一次使用时，可以通过 --help命令查看使用帮助文档。
	当客户端以后台模式运行时，输出运行日志（--help可查看），正常情况下，直接打印在终端无运行日志。
